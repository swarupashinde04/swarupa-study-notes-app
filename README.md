# Swarupa's Study Notes App

A simple web application built using **Python and Flask** that helps organize and manage study notes in an easy and practical way.

This project was created as part of my learning journey while exploring **Python web development**.

---

## Features

* Add new study notes
* Edit existing notes
* Delete notes
* Clean and simple user interface
* Organized card-style notes layout
* Separate Home and Notes pages

---

## Tech Stack

* **Python**
* **Flask**
* **HTML**
* **CSS**

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/swarupa-study-notes-app.git
```

2. Navigate to the project folder

```
cd swarupa-study-notes-app
```

3. Install Flask

```
pip install flask
```

4. Run the application

```
python app.py
```

5. Open in browser

```
http://127.0.0.1:5000
```
